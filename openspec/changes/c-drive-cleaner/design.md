## Context

面向非技术用户的 Windows C 盘清理桌面应用。用户不需要了解系统文件结构，打开软件点几下就能安全释放 C 盘空间。项目从零开始新建，无遗留代码。

## Goals / Non-Goals

**Goals:**
- Electron + React + Ant Design 桌面应用，单 exe 可分发
- 极简模式：一键扫描 → 一键清理，全程最多 2 次点击
- 高级模式：逐项清理 + 大文件分析 + AI 建议
- 内置规则引擎，离线判断每条清理项的安全等级（🟢/🟡/🔴）
- 可选 AI 增强：兼容 OpenAI 格式 API（预置 DeepSeek / MiniMax / 硅基流动）
- 所有删除默认移入回收站，可恢复
- 硬编码排除系统关键目录，杜绝误删

**Non-Goals:**
- 不支持 macOS / Linux（仅 Windows）
- 不做注册表清理（风险高、收益低）
- 不做实时监控 / 后台驻留
- 不做磁盘碎片整理

## Decisions

| 决策 | 选择 | 理由 | 备选方案 |
|------|------|------|---------|
| 前端框架 | React + Ant Design | 中文生态好、组件丰富、社区最大 | Vue 3 + Naive UI / 原生 HTML |
| 架构模式 | Main Process 做文件操作 + Renderer 做 UI | 安全隔离，Node.js 访问系统资源 | 纯 Node.js CLI + 前端静态页 |
| AI 接入 | 兼容 OpenAI 格式 | 一次接入选 DeepSeek/MiniMax/自定义都可用 | 单独集成每家 SDK |
| 文件删除 | 先移回收站 | 用户可恢复，降低误删恐惧 | 直接永久删除 |
| 扫描方式 | 主进程同步遍历 + IPC 进度上报 | 避免大量文件操作阻塞 UI | Web Worker（不适用于 Node 端） |
| 状态管理 | React Context + useReducer | 项目规模不大，无需 Redux 等重型方案 | Redux / Zustand |

## Risks / Trade-offs

- [大文件扫描耗时] → 首次扫描异步进行，显示进度条，用户可随时取消
- [AI 分析泄露隐私] → 仅发送文件名+大小+路径摘要，具体文件内容不发送；在设置中明确提示数据去向
- [系统还原点创建失败] → 独立为可选步骤，静默失败不影响清理流程
- [安全等级误判] → 内置排除列表+白名单双重校验；🟢/🟡/🔴 三级分级，高风险项默认不勾选且需二次确认
- [打包体积过大] → Electron 本身 ~150MB，通过 ASAR 压缩、按需打包、移除平台无关文件控制体积
