# C 盘智能清理工具 — 设计文档

## 概述

面向非技术用户的 Windows C 盘清理桌面应用，提供一键式安全清理 + AI 辅助分析能力。

## 技术栈

| 层面 | 技术选型 |
|------|---------|
| 框架 | Electron |
| 前端 | React + Ant Design |
| 图表 | ECharts |
| 构建 | electron-builder |
| AI | 兼容 OpenAI 格式 API（DeepSeek / MiniMax / 自定义） |

### Main Process 模块

#### 扫描引擎
- 递归扫描预定义的缓存目录列表
- 大文件扫描（递归遍历用户盘，默认排除系统目录）
- 遵循排除列表，绝不扫描列入黑名单的系统路径
- 实时报告扫描进度

#### 规则引擎
- 离线运行，不依赖网络
- 每条扫描结果标注安全等级：
  - 🟢 安全 — 可放心清理
  - 🟡 需注意 — 建议用户确认
  - 🔴 建议保留 — 不推荐清理
- 硬编码排除清单（Windows 系统目录、关键文件类型 .sys 等）

#### AI 模块
- 无 API Key 时跳过，完全由规则引擎工作
- 有 API Key 时将扫描摘要发送给 AI 获取分析建议
- 支持兼容 OpenAI 格式的任意 API（DeepSeek / MiniMax / 自定义）
- 预置提供商配置（Endpoint + 默认模型自动填充）
- AI 助手页面支持自然语言对话

#### 文件操作层
- 所有删除操作默认移至回收站
- 清理前可选创建系统还原点（Windows SystemRestore API）
- 操作日志记录（时间、路径、操作类型）

### Renderer 页面结构

#### 极简模式（默认首页）
- 大按钮「开始扫描」
- 扫描进度动画 + 实时数据
- 结果页：「发现 XX GB 可清理」+「一键清理」
- 完成页：释放空间展示 + 庆祝动效 +「再扫一次」
- 底部入口：「进入高级模式」

#### 高级模式
- 空间概览（ECharts 饼图/条形图 + Top 10 占用）
- 逐项清理（列表 + 安全等级 + AI 建议气泡）
- 大文件分析（排序 + 类型筛选 + 预览）
- AI 助手（对话界面）

## 清理范围

### A — 基础安全清理
| 扫描项 | 规则 | 安全等级 |
|--------|------|---------|
| `%TEMP%` 用户临时文件 | 非进程占用的 .tmp 文件 | 🟢 安全 |
| 浏览器缓存（Chrome/Edge/360） | 超过 7 天的缓存 | 🟢 安全 |
| 回收站 | 全部内容 | 🟢 安全 |
| 缩略图缓存 | 全部 | 🟢 安全 |
| Windows 更新缓存 | 不再需要的更新文件 | 🟢 安全 |

### C — 国产软件缓存
| 扫描项 | 规则 | 安全等级 |
|--------|------|---------|
| 微信缓存 | 非运行进程的缓存 | 🟢 安全 |
| QQ 缓存 | 非运行进程的缓存 | 🟢 安全 |
| 钉钉缓存 | 非运行进程的缓存 | 🟢 安全 |
| 其他国产软件缓存 | 按规则匹配 | 🟡 需注意 |

### D — 大文件分析
- 递归扫描用户指定目录
- 默认排除 `C:\Windows`、`C:\Program Files`、`C:\Program Files (x86)` 等
- 按文件体积降序排列
- 按类型筛选（视频/压缩包/日志/ISO 镜像等）
- 仅提供勾选删除，不移至回收站（大文件可选确认弹窗）

## 安全设计

1. **排除列表** — 硬编码系统目录黑名单，扫描和删除时双重校验
2. **回收站优先** — 所有清理项默认移至回收站，提供恢复引导
3. **还原点** — 清理前可选创建系统还原点，默认开启
4. **操作日志** — 每次清理记录路径、时间、操作类型，保存至本地 `logs/`
5. **二次确认** — 🔴 级别项目即使手动勾选也会弹出确认框

## AI 配置

### 预置提供商

| 提供商 | Endpoint | 默认模型 |
|--------|----------|---------|
| DeepSeek | `https://api.deepseek.com` | `deepseek-chat` |
| MiniMax | `https://api.minimax.chat/v1` | `minimax-text-01` |
| 硅基流动 | `https://api.siliconflow.cn/v1` | `Qwen/Qwen2.5-7B-Instruct` |

### 配置界面
- 选项：不启用 / 预置提供商 / 自定义 OpenAI 兼容
- 预置模式：选择提供商 → 填写 API Key → 自动填充 Endpoint 和模型
- 自定义模式：手动填写 Endpoint、API Key、模型名
- 「测试连接」按钮验证配置有效性

## 数据流

```
用户点击扫描
  → Renderer 通过 IPC 调用 main 进程扫描引擎
  → 扫描引擎遍历目录 + 大文件遍历
  → 每条结果传入规则引擎判断安全等级
  → (可选)汇总发送给 AI 模块获取额外建议
  → 结果通过 IPC 返回 Renderer
  → Renderer 渲染图表 + 列表

用户点击清理
  → Renderer 发送待清理列表到 main 进程
  → 文件操作层：创建还原点(可选) → 移入回收站
  → 写操作日志
  → 返回清理结果
```
